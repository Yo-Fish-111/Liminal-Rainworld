# Contributing

Before contributing, you will need to fork the repository (unless you are an official contributor). You should then commit to a branch named according to the set of changes. Open a pull request when your branch is ready to be merged

## Adding Rooms
In general multiroom dead ends are to be avoided unless something "special" is at the end. 
Try to have many open connections to allow others to add more paths to the region.
Don't be afraid to add something a bit weirder

## Other Contributing 
if you want to contribute code wise we apreaciate it deeply just be careful not to mess up the mod or anyone elses work
